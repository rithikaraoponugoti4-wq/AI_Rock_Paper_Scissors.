# AI Rock Paper Scissors Game

An interactive AI-based Rock Paper Scissors game developed using Python.

## Features
- Play Rock Paper Scissors against the computer
- AI-based random move generation
- Score tracking system
- Multiple rounds gameplay
- User-friendly terminal interaction
- Winner announcement after each round

## Technologies Used
- Python
- Random Module
- Conditional Statements
- Loops
- Functions

## How It Works
1. User enters:
   - Rock
   - Paper
   - Scissors

2. Computer randomly selects a move.

3. Game compares both moves and decides:
   - Win
   - Lose
   - Draw

4. Scores are updated automatically.
